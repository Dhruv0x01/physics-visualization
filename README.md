## 🎯 Key Concept Implemented

### Projectile Motion

* Horizontal motion:
  x = v cos(θ) t

* Vertical motion:
  y = v sin(θ) t − (1/2) g t²

These equations are computed using NumPy and visualized using Matplotlib.
